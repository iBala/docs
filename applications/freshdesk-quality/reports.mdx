# Quality Assessment Reports

Track and analyze your customer service performance.

## Available Reports

<CardGroup cols={2}>
  <Card title="Quality Scores" icon="star">
    Overall and per-category quality metrics
  </Card>
  <Card title="Agent Performance" icon="user">
    Individual agent quality trends
  </Card>
  <Card title="Trend Analysis" icon="chart-line">
    Quality metrics over time
  </Card>
  <Card title="Category Breakdown" icon="folder-tree">
    Performance by ticket category
  </Card>
</CardGroup>

## Analysis Features

<Steps>
  <Step title="View Reports">
    Access comprehensive analytics:
    - Quality scores
    - Performance trends
    - Agent metrics
    - Category analysis
  </Step>

  <Step title="Filter Data">
    Available filters:
    - Date range
    - Agents
    - Categories
    - Teams
  </Step>

  <Step title="Export Results">
    Export options:
    - CSV for spreadsheets
    - PDF for presentations
    - API for integration
  </Step>
</Steps>
