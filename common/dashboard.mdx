## Using Your Agents

### Quick Start
1. Find your agent in:
   - Recently Used section
   - Hired Agents tab
2. Click "Open" to launch
3. Start working in the agent's workspace

### Agent Workspace
Each agent provides:
- Dedicated configuration panel
- Task management interface
- History and analytics
- Integration options
